 Key Papers to Include in Your Literature Survey

    SRCNN
    Paper: Image Super-Resolution Using Deep Convolutional Networks (Dong et al., 2014)
    ➤ First deep learning-based SR model, using a simple 3-layer CNN. Introduced end-to-end learning for SR.

    VDSR
    Paper: Accurate Image Super-Resolution Using Very Deep Convolutional Networks (Kim et al., 2016)
    ➤ Used a very deep network and residual learning to enhance accuracy and reduce training difficulty.

    ESPCN
    Paper: Real-Time Single Image and Video Super-Resolution Using an Efficient Sub-Pixel Convolutional Neural Network (Shi et al., 2016)
    ➤ Introduced efficient upsampling using sub-pixel convolutions, enabling real-time performance.

    SRGAN (Core Paper)
    Paper: Photo-Realistic Single Image Super-Resolution Using a Generative Adversarial Network (Ledig et al., 2017)
    ➤ Introduced GANs for SR and perceptual loss using VGG features for photorealistic image enhancement.

    EnhanceNet
    Paper: EnhanceNet: Single Image Super-Resolution through Automated Texture Synthesis (Sajjadi et al., 2017)
    ➤ Alternative GAN-based SR model with texture synthesis and perceptual loss.

    ESRGAN
    Paper: ESRGAN: Enhanced Super-Resolution Generative Adversarial Networks (Wang et al., 2018)
    ➤ Improved version of SRGAN with a better perceptual loss and network structure.

    Real-ESRGAN
    Paper: Real-ESRGAN: Training Real-World Blind Super-Resolution with Pure Synthetic Data (Wang et al., 2021)
    ➤ Addressed real-world noise and degradation, making GAN-based SR practical in non-ideal scenarios.
